How to Use the App

Follow these steps to use the CGPA calculator:

  1. Enter the number of courses you want to calculate the CGPA for.
  2. For each course, enter the course code, score (between 0 and 100), and weight unit.
  3. Click the "Calculate CGPA" button to generate the results table.
  4. The results table will display the course details, including the course code, score, grade, weight unit, grade point, and the calculated CGPA.

Algorithm

The algorithm for the CGPA calculation program follows these steps:

  1. Input the number of courses.
  2. Initialize arrays for course codes, scores, grades, grade points, weight units, and weighted grade points.
  3. Collect course codes, scores, grades, and weight units for each course.
  4. Calculate weighted grade points and totals.
  5. Calculate CGPA.
  6. Prepare and display the results table.
